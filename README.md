# Camel

Agent-based knowledge management system. AI agents are first-class citizens — users are expected to interact with agents more than the editor. Interaction isn't limited to the app; API, Telegram, email, and Slack are treated as core surfaces.

## Setup

Requires Node.js 20+, npm, and the [Encore CLI](https://encore.dev/docs/install).

```bash
cd backend
npm install
```

`npm install` bootstraps secrets and initializes the local database automatically.
